## 🫧 Bubble Pop Blast! — HTML5 Game

A complete, browser-based **Bubble Shooter** game built for platforms like
[CrazyGames](https://www.crazygames.com) and [Poki](https://poki.com).
Enjoyed by both boys and girls — colourful, fast-paced, and seriously addictive!

### 🎮 How to Play

1. Open `index.html` in any modern browser (no build step needed).
2. Aim with your mouse or finger, then click/tap to shoot a bubble.
3. Match **3 or more** bubbles of the same colour to pop them.
4. Floating bubbles fall for bonus points!
5. Clear the entire board to complete each level.

### ✨ Features

| Feature | Details |
|---|---|
| 🎯 10 Levels | Progressive difficulty — from easy 4-colour grids to 6-colour boss levels |
| 💰 Coin System | Earn coins for pops, combos & level clears; spend in the Shop |
| 🛒 Shop | Buy **Bomb**, **Rainbow**, **Freeze** & **Extra Life** power-ups |
| ❤️ Lives | 3 lives per run; replenishable with an Extra Life power-up |
| 🏆 Leaderboard | Top 10 scores saved locally |
| 🎵 Procedural Music | Background melody + SFX via Web Audio API |
| 📱 Responsive | Works on desktop & mobile |

### 📁 File Structure

```
index.html          ← game entry point
css/
  style.css         ← all styling & animations
js/
  config.js         ← constants & 10 level definitions
  storage.js        ← localStorage persistence
  audio.js          ← Web Audio API sounds & music
  game.js           ← complete game engine
```

### 🚀 Quick Start

Just open `index.html` in your browser — no server required for local play.
For hosting on CrazyGames / Poki, upload all files as-is; the game is fully
self-contained with no external dependencies.
